What is {% app_name %}?

{% app_name %} is a tool for accessing open government data. It helps you find and download collections of data published by government agencies, research institutions, and other organizations. Once you find the data you need, you can preview it using maps, graphs and tables - whether you are a developer, journalist, researcher, NGO, citizen, or government staff member.

{% app_name %} offers a central point of access to open, freely accessible data from public administration ("open government data", or OGD). You can find more information here.

What are datasets and resources?

For {% app_name %} purposes, data is published in units called "datasets". A dataset is a parcel of data - for example, it could be crime statistics for a region, spending figures for a government department, or temperature readings from weather stations.

A dataset contains two things:

  • Information or "metadata" about the data: The title and publisher, date, what formats it's available in, what license it's released under, etc.
  • A number of "resources": These hold the actual data files. {% app_name %} can handle data in any format - CSV, Excel spreadsheets, XML files, PDF documents, image files, etc. A dataset can contain any number of resources, for example, different files might contain data for different years, or the same data in different formats.

How do I search for data?

To find datasets, type any combination of search words (e.g. "health", "transport", etc.) in the search box on any page. You can:

  • Browse results: View multiple pages of search results
  • Refine your search: Modify your search terms and try again
  • Use filters: Restrict results by tags, data formats, organizations, topics, or licenses using the filters in the left column
  • Combine filters: Add and remove multiple filters to narrow down results

You can also search within a specific organization by visiting their page and using the search box there.

What file formats are supported?

{% app_name %} supports a wide variety of data formats including:

  • Tabular data: CSV (comma-separated values), Excel (XLSX), TSV (tab-separated values)
  • Structured data: JSON, XML
  • Documents: PDF, Word documents, HTML
  • Images: Common image formats (JPG, PNG, etc.)
  • Geographic data: Geospatial formats for mapping
  • Other formats: Various government and industry-standard data formats

Can I preview data files before downloading?

Yes! {% app_name %} offers preview functionality for many file types:

  • CSV, Excel, and TSV files: View data in a formatted table with the first several rows displayed
  • JSON and XML files: View structured data with proper formatting
  • PDF and HTML files: Preview document content
  • Images: View image files directly in the browser

Look for the "Preview" button next to downloadable resources. Preview shows a sample of the data to help you determine if it meets your needs before downloading.

How do I download datasets?

To download data:

  1. Find the dataset you're interested in using the search function
  2. Navigate to the dataset page by clicking on its title
  3. Browse the available resources (files) listed for that dataset
  4. Click the "Download" button next to the resource you want
  5. The file will be downloaded to your device

Each dataset may contain multiple resources in different formats - choose the format that best suits your needs.

What are organizations and topics?

Organizations represent the government agencies, departments, or institutions that publish data on the portal. Each dataset is owned by one organization. You can browse datasets by organization to find data from specific government bodies. Each organization has its own page where you can find information about the organization and search within its datasets.

Topics are subject categories that help organize datasets by theme or area of interest, such as environment, economy, health, education, etc. Topics make it easier to find related datasets across different organizations, allowing you to discover data on specific themes regardless of which agency published it.

Do I need an account to access data?

No, registration is not needed to search for and download public data. You can browse all publicly available datasets and download files without creating an account.

What can I do with an account?

Creating a free account provides additional features:

  • Follow datasets: Get notified when datasets you're interested in are updated
  • Follow organizations: Receive notifications about new datasets from specific agencies
  • Personal dashboard: View your news feed with updates on datasets and organizations you follow
  • User profile: Manage your personal information and preferences
  • Enhanced personalization: Customize your experience on the platform

What licenses apply to the data?

Each dataset comes with specific licensing information that defines how you can use the data. Common licenses include:

  • Open licenses: Allow free use, sharing, and modification with minimal restrictions
  • Attribution licenses: Require crediting the data source when you use the data
  • Restricted licenses: May have limitations on commercial use, redistribution, or modification

License information is displayed on each dataset page. Always review the license terms before using data to ensure you comply with the usage requirements.

How can I contact data publishers?

Each dataset page includes contact information for the publishing organization. You can:

  • Dataset-specific contacts: Look for author or maintainer contact information on individual dataset pages
  • Organization contacts: Visit the organization's page for general contact details
  • General inquiries: Use our contact form if you need help reaching a data publisher

How do I use advanced search features?

{% app_name %} supports both simple and advanced search:

Simple Search:

  • Type words or phrases in the search box
  • Use +word to require a word (mandatory)
  • Use -word to exclude a word (prohibited)
  • Use "exact phrase" to search for exact phrases

Advanced Search (using field:term syntax):

  • title:health - search for "health" in dataset titles
  • title:health* - search for words starting with "health"
  • author:smith - search for datasets by author "smith"
  • text:environment - search across all text fields

How often is data updated?

Update frequency varies by dataset and depends on the publishing organization. Each dataset page shows:

  • Last updated: When the data was most recently modified
  • Update frequency: How often the publisher typically updates the data (if specified)
  • Data coverage period: The time period the data represents
  • Activity stream: History of recent changes to the dataset

For questions about specific dataset update schedules, contact the publishing organization directly using the contact information provided on the dataset page.
